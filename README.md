# My-BaiduIME-Skin
---
我的百度输入法安卓皮肤，用了许多年，原版是Ray制作的apple风格划选布局，是一款效率非常高的皮肤，因为自己对按键映射和UI稍微做过一些修改，换手机容易丢掉皮肤文件，遂在此备份。

### 皮肤存放位置：
`sdcard0/baidu/ime/skins`
